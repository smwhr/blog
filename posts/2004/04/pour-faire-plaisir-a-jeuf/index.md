Supposons que ln(-1) existe.

on a ln(-1) x ln(1) = ln(-1) x 0 = 0  
et ln(-1) x ln(1) = ln(-1+1)=ln(0) = -infinity  

On obtient une contradiction.

Donc ln(-1) n'existe pas.

Donc on peut l'inventer.

Je vous remercie.
